# Triage Labels

**Not used in this repo.**

Issue tracking is on GitHub without Matt Pocock triage labels. Skills that mention roles like `ready-for-agent` or `needs-triage` should **not** create or apply labels here — see `issue-tracker.md`.

If you later want triage workflow, add the five canonical labels on GitHub and update this file:

| Role | Label |
| --- | --- |
| needs-triage | `needs-triage` |
| needs-info | `needs-info` |
| ready-for-agent | `ready-for-agent` |
| ready-for-human | `ready-for-human` |
| wontfix | `wontfix` |
