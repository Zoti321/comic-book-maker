# 添加 Page Image

## What to build

用户打开 Project 后，通过平台文件选择器选取 Page Image，Core 将文件复制到 Project Storage 的 `assets/`，在 Library Database 中写入 Asset Reference 与页序。Flutter 编辑器展示 Page 列表（v1 可用原图缩略展示）。

## Acceptance criteria

- [ ] 支持 Page Image 格式：jpg、jpeg、png、webp、gif、avif、bmp
- [ ] 不支持格式被拒绝并给出可理解错误提示
- [ ] 选图后 Page 按添加顺序出现在编辑器列表中
- [ ] 图片文件存在于 Project Storage `assets/`，Library Database 存相对路径引用
- [ ] 可一次选择多张图片批量添加
- [ ] Create Project → 添加 Page 的完整路径可在 UI 上走通
